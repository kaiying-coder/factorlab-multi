# FactorLab — 多因子量化研究平台

A 股多因子分层回测研究平台:从因子构建、IC/IR 分析、交易成本测算、多因子组合,到 LLM + RAG 自动生成投研备忘录,并部署为可交互看板。

> 个人学习项目。数据为 A 股公开日行情衍生数据,代码与方法自行实现。

**在线 Demo**:https://factorlab-multi-p2qmcavx5hdgepsq9jgpw7.streamlit.app

---

## 项目概览

本项目实现了一套完整的多因子研究流程,分四层:

| 层 | 内容 |
|---|---|
| 数据层 | A 股 2006–2025 全市场日收益面板(约 5300 只股票、4700+ 交易日、1258 万行) |
| 金融工程层 | 分层回测、IC/ICIR 分析、交易成本/换手率、因子相关性与复合因子 |
| AI 层 | LLM 生成投研备忘录 + RAG 检索增强(研报向量化检索 + 回测数据) |
| 展示层 | Streamlit 交互看板,部署上线 |

## 因子与主要结果

测试四个因子(均从日收益或市值排名构建):**市值、动量、反转、低波动**。

基于 A 股 2006–2025 全样本(分 10 组、日度调仓)的主要结果:

| 因子 | IC 均值 | ICIR | 多空年化 | 多空 Sharpe |
|---|---|---|---|---|
| 反转 reversal | +0.055 | 0.37 | 79.0% | 3.92 |
| 动量 momentum | −0.043 | −0.28 | 57.4% | 2.68 |
| 低波动 volatility | +0.035 | 0.21 | 11.1% | 0.50 |
| 市值 market_cap | −0.018 | −0.12 | 36.5% | 1.75 |
| **复合因子** | — | — | **86.1%** | **4.05** |

几个发现:

- **反转因子在 A 股最强**(ICIR 0.37),符合散户主导、短期超跌反弹显著的市场特征。
- **动量因子呈反向**(ICIR −0.28),与美股的正向动量相反。
- **交易成本影响显著**:反转因子日换手率约 69%,扣双边 0.3% 成本后多空年化由 79% 降至 69%、Sharpe 由 3.92 降至 3.44。
- **多因子组合优于单因子**:动量与反转 IC 相关性 −0.62(强负相关),按 ICIR 加权合成的复合因子 Sharpe 达 4.05,优于全部单因子。

## 方法要点

- **防前视偏差**:因子值用 `shift(1)`(仅含截止上一日信息),收益用次日对齐(`shift(-1)`),因果方向严格分离。
- **IC**:使用 RankIC(Spearman),比 Pearson 更稳健;ICIR = IC 均值 / IC 标准差,衡量预测稳定性。
- **多空方向**:跟随 IC 符号自动确定(IC>0 做多高分位组,IC<0 反之),避免方向写死导致的错误。
- **复合因子**:横截面 z-score 标准化 → 按 IC 符号统一方向 → 按 |ICIR| 加权求和。
- **RAG**:研报文档向量化建库 → 按查询检索 top-k → 检索结果与回测事实共同输入 LLM;通过 prompt 约束区分"数据事实"与"外部观点"以降低幻觉。

## 说明与局限

- 在线看板使用预计算的固定结果(10 组、全样本),通过下拉切换因子查看细节。
- 备忘录的 LLM 生成需配置 API key;线上为预生成的只读展示,不实时调用模型(避免公开应用产生 API 费用)。
- 本项目用于方法学习与展示,**未做行业/市值中性化**,因子收益可能混杂行业暴露;交易成本为简化假设;**不构成任何投资建议**。
